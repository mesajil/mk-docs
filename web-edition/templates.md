

# PC Case

## Características destacadas

* Marca
* Modelo
* Factor de forma: Full, Mid o Mini Tower
* Puertos principales
* RGB integrado
* PSU integrado
* Tiene estructura abierta
* Material y aleación de aluminio

## Complementos de nombre

Complementan el nombre del producto.

* `ARGB`: RGB Integrado
* `LCD`: LCD integrada
* `[COLOR]`: Puede ser `BLACK`, `WHITE`, `SILVER`, etc.
* `(BTF)`
* `(Xin1) CASE + FUENTE + Cooler CPU + Soporte GPU`: Puede ser `(3IN1)`, `(4IN1)`, etc.

## Ejemplo de complementos de nombre:

CASE con ARGB, LCD Integrado y BTF. 

```text
ARGB LCD (BTF)
```

## Ejemplo de complementos de nombre:

CASE White Edition 3 en 1: Fuente y Cooler CPU integrados. 

```text
WHITE (3IN1) CASE + FUENTE + Cooler CPU
```
